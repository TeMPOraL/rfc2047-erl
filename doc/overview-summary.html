Welcome to the RFC1342 Decoder library!

Authors: Jacek Zlydach (jacek.zlydach@erlang-solutions.com).

RFC1342 Decoder is a tool for processing strings in e-mail headers that can contain different charsets and encodings next to each other in the same text. It uses erlang-iconv for charset conversions.
